#   Library-Management-System

Python-based **Library Management System** using CSV file handling.

---

## 🔹 Introduction

This project is a console-based Library Management System developed in Python. It helps manage library operations using CSV file storage.

---

## 🔹 Objective

To automate basic library operations like adding, searching, issuing, and returning books.

---

##  🔹 Tools & Technologies

- Python 3  
- CSV Module  

---

## 🔹 Features

-  Add new books  
-  Search books  
-  Issue books  
-  Return books  
-  Display reports  

---

## 🔹 Working

The system stores all data in a CSV file.  
When a book is issued, quantity decreases, and when returned, quantity increases.

---

## 🔹 Conclusion

This project demonstrates core Python concepts such as file handling, loops, and functions in a real-world application.
